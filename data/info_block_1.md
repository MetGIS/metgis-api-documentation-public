# Contents of Object Info

The JSON Object Info contains general information about the weather data. The following fields are included:
- [Description](variable_list_point_api.md#description)
- [Forecast_Calculated_LocalTime](variable_list_point_api.md#forecast_calculated_local_time)
- [Forecast_Calculated_UTC](variable_list_point_api.md#forecast_calculated_utc)
- [Latitude](variable_list_point_api.md#latitude)
- [Longitude](variable_list_point_api.md#longitude)
- [Altitude](variable_list_point_api.md#altitude)
